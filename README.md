# YassirChallengeTask
IOS Movie App written in SwiftUI using MVVM

## Requirements
- Swift 5.0
- Xcode 14.0+
- iOS Target 16.4+

## Installation

* Clone the project.
* add .env file on the project's root, .env file contains the movieApiKey[will be shared over the email]
* Run the project
